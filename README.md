# CineMatch 🎬

CineMatch is a modern AI-inspired movie recommendation web application that helps users discover movies based on their mood, genre, vibe, language, favorite actors, directors, and personal preferences.

## Features
- 🎥 Smart movie recommendation system
- 🎭 Genre and vibe selection
- 🌍 Multiple language support
- ⭐ Actor and director preference filtering
- 🚫 Avoid unwanted categories
- ✨ Beautiful animated UI
- 📱 Fully responsive design

## Technologies Used
- HTML5
- Tailwind CSS
- JavaScript
- Node.js Concepts

## How It Works
Users enter their movie preferences, and the system recommends movies that best match their interests using intelligent filtering and recommendation logic.

## Project Purpose
This project was developed as an Introduction to AI mini project to demonstrate recommendation systems and personalized user experiences.
